Практические ноутбуки по Apache Spark на реальных задачах обработки данных с использованием объектного хранилища S3 (Yandex Cloud / AWS).

## Содержание

Репозиторий включает Jupyter Notebook-и с решениями типовых задач Data Engineer:

- Работа с DataFrame API и Spark SQL
- Join различных типов, агрегации, оконные функции
- Чтение и запись данных в S3 (Parquet, CSV, JSON)
- Партиционирование данных для оптимизации запросов
- Преобразование временных зон (`to_utc_timestamp`)
- Обогащение данных через объединение справочников
